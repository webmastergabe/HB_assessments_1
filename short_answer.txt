8) What is the difference between `git add` and `git commit`?

Your answer: 'git add' is a way of saving and tracking code on individual files as you are working. 'git commit' creates a backup of your file and will create a "version" that can be viewed again.


------------------
9) What does the `git push` command do?

Your answer: 'git push' allows user to upload (or 'push') the project to a remote repository like on GitHub.


------------------
10) When would you use the `git clone` command?

Your answer: 'git clone' is used if you have worked on your repo on a different device but want the changes synced and copied to your current device.
